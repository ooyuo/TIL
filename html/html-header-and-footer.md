# header와 footer

## 1. `<header>`

`<header>`는 소개나 탐색을 돕는것의 그룹을 나타낸다. 일부 제목 요소, 로고, 구획의 제목, 탐색 폼과 같은 것들이 포함된다.

`<header>` 태그안에 `<header>`, `<footer>`, `<address>` 는 후손이 될 수 없다.



### 1.1 `<header>` 사용법

```javascript
<header>
	a logo
</header>
```





## 2. `<footer>`

`<footer>` 는 일반적으로 작성자 구획, 저작권 데이터, 관련된 문서의 링크에 대한 정보를 포함한다.

`<footer>` 태그안에 `<header>`, `<footer>`, `<address>` 는 후손이 될 수 없다.



### 1.1 `<header>` 사용법

```javascript
<header>
	a logo
</header>
```
