# gitignore 작성 규칙에 대해서

#### `**.gitignore`란?**

Git 버전 관리에서 제외할 파일 목록을 지정하는 파일이다.

ex) Backup File이나 Log File 등..

 

 

사용 방법

1. `.gitignore`라는 이름의 파일을 만든다.

2. https://github.com/github/gitignore/blob/master/Node.gitignore 접속하여 코드를 복사해 붙여넣는다.

+ 추가로 ignore하고 싶은 파일이 있다면 해당 파일명을 추가로 작성해주면 된다.
